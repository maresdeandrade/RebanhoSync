# Template Codex Task - Genérico

Use este template para definir tarefas para o Codex de forma concisa.

## Tarefa

Descreva o objetivo principal da tarefa em 1 a 3 frases claras.

## Escopo Permitido

Liste as áreas ou arquivos que o Codex pode modificar ou investigar.

## Escopo Proibido

Liste as áreas ou arquivos que o Codex NÃO deve modificar ou investigar (ex: migrations, RLS, seed, refatoração ampla).

## Arquivos Prováveis

Liste os caminhos de arquivos que provavelmente serão afetados pela tarefa.

## Validação

Descreva os comandos ou critérios de validação que devem ser executados ou considerados para verificar a conclusão da tarefa.

## Formato de Resposta

Especifique o formato esperado para a resposta do Codex (ex: resumo executivo, arquivos alterados, riscos, etc.).

## Observações Adicionais

Qualquer informação extra relevante para a execução da tarefa.
