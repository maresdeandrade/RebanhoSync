# Arquivo Historico

Este diretorio guarda materiais preservados apenas como historico tecnico.

## Estrutura

- `analysis/`: analises de dominio e modelagem feitas durante fases anteriores do produto
- `review/`: auditorias e relatorios de revisao que nao participam mais do processo vivo
- `performance/`: baselines e registros pontuais de investigacao

## Uso esperado

- consulte estes arquivos apenas quando precisar recuperar contexto antigo;
- nao trate este diretorio como snapshot do repositorio;
- para o estado atual, use `docs/CURRENT_STATE.md`, `docs/STACK.md`, `docs/ROUTES.md` e `docs/IMPLEMENTATION_STATUS.md`.
