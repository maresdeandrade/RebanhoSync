# Brand Direction — RebanhoSync

Status: Direção visual aprovada  
Tipo: Referência de marca para evolução do produto  
Versão: v1 — Azul Sync Técnico

## Posicionamento

RebanhoSync é uma ferramenta simples, operacional e offline-first para gestão de pecuária de corte.

A marca deve transmitir:

- confiança;
- precisão;
- operação em campo;
- sincronização segura;
- tecnologia útil;
- clareza para tomada de decisão.

## Conceito visual

Direção: Azul Sync Técnico.

A identidade combina:

- azul petróleo;
- estrutura limpa;
- cards operacionais;
- ícones lineares;
- estados claros de sincronização;
- símbolo técnico com boi em escudo/hexágono.

## Logo

Usar símbolo em formato de escudo/hexágono técnico, com cabeça de boi branca e elementos discretos de sincronização/circuito.

Evitar:

- brinco/etiqueta como símbolo principal;
- foto de animal;
- boi hiper-realista;
- folha genérica;
- nuvem/Wi-Fi como elemento principal.
