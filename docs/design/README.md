# Design — RebanhoSync

Status: Direção visual em validação  
Escopo: Identidade visual, referências de UI e prompts de geração visual  
Produto: RebanhoSync — gestão simples e offline-first para pecuária de corte

## Objetivo

Esta pasta consolida a direção visual do RebanhoSync para orientar refinamentos de UI, geração de telas de referência e futuras implementações em React/Tailwind.

Estes documentos não substituem o código atual, não alteram regra de negócio e não autorizam criação de funcionalidades fora do escopo do produto.

## Documentos

- `BRAND_DIRECTION.md` — direção de marca, logo, paleta, tipografia e iconografia.
- `UI_VISUAL_REFERENCES.md` — telas de referência aprovadas como direção visual.
- `STITCH_PROMPTS.md` — prompts para gerar novas variações visuais no Stitch.
- `references/` — imagens de referência.

## Princípios

- Mobile-first.
- Offline-first.
- Clareza operacional.
- Identidade azul “Sync Técnico”.
- Cards brancos em fundo claro.
- Ícones lineares estilo Lucide.
- Sem foto real de animal como avatar padrão.
- Sem funcionalidades não existentes.

## Limites

Estas referências não são pixel-perfect.

Não usar estes documentos para criar:

- ERP fiscal;
- marketplace;
- venda/abate;
- carência sanitária conclusiva;
- aptidão comercial;
- IA preditiva;
- novas regras de negócio.
